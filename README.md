# detect-color-opencv

Virtual Env commands

- Creating - ```python -m venv venv```
- Activating - ```venv\Scripts\activate.bat```
- Listing - ```pip list```
- Saving - ```pip freeze > requirements.txt```
- Restoring - ```pip install -r requirements.txt```
- Deactivating - ```deactivate```
